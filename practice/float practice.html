본래 float 속성을 만들었을 때는 img Tag에 text를 붙이는 것을 기본목적으로 하였다. 그러나 float 속성을 이용해 공간 tag들을 수평절렬하기 시작하면서 해당 용도로 많이 사용됨. 이를 추가 활용하여 webPage Layout을 제작함 하지만 float 속성으로 layout을 설계하기에는 완벽하지 않은 부분들이 있음 1. 공간 = display : block; 은 width : 100%이므로 좌,우를 다 차지함 (자신외 아무것도 올리지 않음) 2. web = 공간들의 배치 > 상,하(X) / 좌,중,우(O) > 위 1번 조건과 상충되는 내용 3. 해결방안 = float, grid, flex float : img 옆에 text 붙이기가 주목적 단점 : 기존 block을 무효화함. 좌,우 서로의 높이가 다를경우 Layout이 무너짐 주요 기능 : float 속성이 적용된 요소 다음에 위치하는 요소들이 float 속성이 적용된 요소와 겹치는 현상 방지 요소에 float 속성이 적용되면 그 이후에 등장하는 모든 요소들은 정확한 위치를 설정하기가 매우 힘들어짐 따라서 clear속성을 사용하여 이후의 요소들이 더는 float속성 영향 받지않도록 설정 (요소들이 float 속성이 적용된 요소 아래에 위치하도록 한다.) clear : both( float : left, float:right 둘다 취소시킴)
