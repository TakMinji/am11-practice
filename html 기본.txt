태그의 중첩을 부모 자식관의 관계라고 볼 수 있다.
태그 간의 관계를 나타내는 
부모(parent)태그, 자식(child)태그


부모태그 ) ul(순서가 없는 리스트), ol(순서가 있는 리스트)
자식태그 ) li
--------------------------------------------------------------------------------------속성
href 속성 
태그 a는 하이퍼링크를 정의
속성 href은 링크가 이동하는 페이지의 url을 지정

src 속성
태그 img는 html 페이지에 이미지를 삽입하는 데 사용됨

alt 속성
alt태그의 필수 속성 img은 어떤 이유로 이미지를 표시할 수 없는 경우 이미지에 대한 대체 텍스트를 지정
src 이는 연결 속도가 느리거나 속성에 오류가 있거나 사용자가 화면 판독기를 사용하는 경우 발생할 수 있음.

속성 값을 묶는 큰 따옴표는 html에서 가장 일반적이지만 작은 따옴표를  사용할 수 있음
일부 상황에서는 속성 값 자체게 큰 따옴표가 포함되어 있으면 작은 따옴표를 사용해야 함.

id 속성
HTML 요소의 고유 ID를 지정(중복 불가)
이 id 속성은 css의 특정 스타일 선언을 가리키는데 사용. 
또는 js에서 특정 요소의 스타일을 지정/선택하는 데 사용
*id 이름은 대소문자를 구분함!
*id 이름은 하나 이상의 문자를 포함해야 하며 숫자로 시작할 수 없으며 공백 포함불가

모든 html 요소는 속성을 가질 수 있음

class와 id의 차이점)
class는 여러번 선언 가능하지만 id는 페이지 내의 하나에 html요소에서만 사용 가능

-------------------------------------------------------------------------------------- 텍스트 형식

<strong>- 중요한 텍스트
<i>- 기울임꼴 텍스트
<em>- 강조된 텍스트 (*언어 강세를 사용하여 단어 발음함)
<mark>- 표시된 텍스트
<small>- 더 작은 텍스트
<del>- 삭제된 텍스트
<ins>- 삽입된 텍스트
<sub>- 아래 첨자 텍스트
<sup>- 위 첨자 텍스트
--------------------------------------------------------------------------------------인용

q 짧은 인용문을 위함

span

display : inline
--------------------------------------------------------------------------------------레이아웃 요소
header - 머리, 문서 또는 섹션의 헤더 정의
nav - 탐색 링크 세트(모든 링크가 nav안에 있어야하는 것은 아님 탐색 링크 주요 볼록에만 사용됨)
section - 문서의 섹션(소개, 콘텐츠, 연락처 정보, 뉴스항목 등 주제가 있는 콘텐츠 그룹)
article - 독립적이고 독립적인 콘텐츠 정의(주변 콘텐츠와 간접적으로 관련되어야 함)
aside - 콘텐츠 이외의 콘텐츠(사이드바)
footer - 문서 또는 섹션의 바닥글 정의
--------------------------------------------------------------------------------------figure 및 figcaption
figure 일러스트레이션, 다이어그램, 사진, 코드목록 등과 같은 독립된 콘텐츠를 지정
figcaption 캡션 정의 (figure의 자손)
--------------------------------------------------------------------------------------양식
form 텍스트 필드, 체크박스, 라디오 버튼, 제출 버튼 등과 같은 다양한 유형의 입력 요소에 대한 컨테이너
input 가장 많이 사용되는 양식 요소
<input type="text">	한 줄 텍스트 입력 필드를 표시
<input type="radio">	라디오 버튼을 표시(여러 선택 항목 중 하나 선택).
<input type="checkbox">	확인란을 표시(여러 선택 사항 중 0개 이상 선택).
<input type="submit">	제출 버튼을 표시(양식 제출용).
<input type="button">	클릭 가능한 버튼을 표시
